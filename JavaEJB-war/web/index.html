Description:
 Project Type: JavaEJB an Enterprise Application in Java

Technology Used: Java Programming Language with Object Oriented Language and Database(JDBC), HTML, JavaScript

Tools: Netbeans, XAMPP, Mysql, Notepadd++

Functionalities: Create Read Update Delete, EJB modules, Servlets, JSP,

GitHub URL: "https://github.com/nishu24/JavaEJB.git"

How to Implement:
- First clean and build all the three modules of the project i.e. EE, Web Application and EJB.
- Then Deploy all those modules again to make the servlets and Java bean functionality work.
- Run the main EE module in project so that it run the main jsp page i.e. "index.jsp"
- Running only the application module will not enable java beans and sevlets to work properly
  as they are present in other parts of the project.
